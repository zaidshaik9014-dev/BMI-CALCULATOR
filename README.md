# BMI-CALCULATOR
A simple C program that calculates Body Mass Index (BMI) using weight and height to determine health category.

A beginner-friendly program to calculate Body Mass Index (BMI).

**Features:**
- Takes weight (kg) and height (m) as input
- Calculates BMI = weight / (height * height)
- Displays category: Underweight, Normal, Overweight, or Obese
- Simple text-based interface

**Language:** C  
**Usage:** Compile and run `bmi_calculator.c`, then enter your weight and height.
